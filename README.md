Dubbo Http Protocol Demo
========================

Run Provider
---------------------

```bash
mvn jetty:run
```

Run Consumer
----------------------

```bash
mvn exec:java -Dexec.mainClass=com.dubbo.demo.consumer.ConsumerMain
```
